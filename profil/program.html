Program PKBM Nakai Cerdas Belajar

Berbagai program pendidikan nonformal yang diselenggarakan untuk memenuhi kebutuhan masyarakat Nakai Cerdas Belajar

Program Keaksaraan

Keaksaraan Dasar

Program untuk masyarakat yang buta aksara, fokus pada kemampuan membaca, menulis, dan berhitung dasar.

  • Durasi: 6 bulan
  • Target: Dewasa buta aksara
  • Metode: Pembelajaran kontekstual

Keaksaraan Lanjutan

Pengembangan kemampuan keaksaraan untuk penerapan dalam kehidupan sehari-hari dan dunia kerja.

  • Durasi: 4 bulan
  • Prasyarat: Lulus keaksaraan dasar
  • Fokus: Aplikasi praktis

Program Kesetaraan

Paket A (Setara SD)

Program pendidikan setara Sekolah Dasar untuk masyarakat yang tidak sempat menyelesaikan pendidikan dasar.

  • 6 tingkat pembelajaran
  • Ujian kesetaraan nasional
  • Ijazah diakui negara

Paket B (Setara SMP)

Program pendidikan setara Sekolah Menengah Pertama dengan kurikulum yang disesuaikan.

  • 3 tahun program
  • Pembelajaran tematik
  • Praktik keterampilan

Paket C (Setara SMA)

Program pendidikan setara Sekolah Menengah Atas dengan peminatan sesuai bakat dan minat.

  • 2 program peminatan
  • Persiapan kuliah/kerja
  • Keterampilan vokasional

Program Keterampilan

Keterampilan Komputer

Pelatihan komputer dasar, office, dan internet untuk meningkatkan digital literacy

Tata Boga

Pelatihan memasak dan pengolahan makanan khas daerah untuk usaha kuliner

Pertanian Organik

Pelatihan pertanian modern dan organik untuk meningkatkan hasil pertanian

Kerajinan Tangan

Pengembangan keterampilan membuat kerajinan dari bahan lokal yang bernilai ekonomi
